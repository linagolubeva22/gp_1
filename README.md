## Описание проекта

В проекте проводится разведочный анализ данных по контенту Netflix.  
Основная задача — изучить, какие характеристики фильмов и сериалов связаны с пользовательской оценкой `user_rating_score`.

Мы анализируем возрастные рейтинги, годы выпуска, тип контента, жанры, награды, выручку, длительность и страну производства.  
Цель анализа — понять, какой контент потенциально лучше подходит пользователю и какие признаки могут быть полезны для дальнейшей рекомендательной модели.

**Целевая переменная:** `user_rating_score`  
Она рассматривается как показатель того, насколько контент подходит конкретному пользователю.

## Данные

### 1. Исходный датасет Netflix Shows

В качестве основного источника используется датасет **1000 Netflix Shows** с Kaggle.  
Датасет содержит информацию о фильмах и сериалах Netflix по состоянию на 2017 год.

**Основные признаки:**

| Признак | Описание |
|---|---|
| `title` | Название фильма или сериала |
| `rating` | Возрастной рейтинг: G, PG, TV-14, TV-MA и др. |
| `ratingLevel` | Текстовое описание возрастного рейтинга |
| `ratingDescription` | Числовое описание рейтинга |
| `release year` | Год выпуска контента |
| `user rating score` | Пользовательская оценка / скор рекомендации |
| `user rating size` | Дополнительный показатель пользовательского рейтинга |

**Файл:** `NetflixShows.xlsx`

---

### 2. Обогащенный датасет

Для более глубокого анализа был использован дополнительный датасет с новыми признаками.

**Добавленные признаки:**

| Признак | Описание |
|---|---|
| `genres` | Жанры контента |
| `fees` | Выручка фильма / проекта |
| `showLength` | Длительность: минуты для фильмов, количество серий для ТВ-шоу |
| `premiere` | Дата премьеры |
| `awards` | Количество наград |
| `countries` | Страна производства |
| `true_type` | Реальный тип контента: Movie или TV Show |

**Файл:** `data.xlsx`

## Что было сделано

В рамках проекта выполнены следующие этапы:

1. Первичный обзор исходного датасета.
2. Анализ и удаление дубликатов.
3. Обработка пропусков.
4. Очистка нерелевантных признаков.
5. Создание новых признаков:
   - `rating_type` — TV / not TV;
   - `age_group` — возрастная категория;
   - `release_year_category` — временной период выпуска;
   - `awards_true` — наличие или отсутствие наград.
6. Анализ распределения рейтинговых групп.
7. Анализ пользовательских оценок.
8. Анализ влияния возраста контента на оценку.
9. Анализ различий между фильмами и сериалами.
10. Анализ жанров.
11. Анализ влияния наград, выручки и длительности.
12. Построение визуализаций.
